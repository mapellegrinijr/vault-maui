# .maui

Estado tecnico do Maui.

Esta pasta deve conter metadados tecnicos, estado operacional e configuracoes internas do Maui.

Nao usar `.saara/` como estado tecnico do Maui. Caso `.saara/` exista, ela pertence ao Saara e deve ser preservada.

## Git

O repositorio Git do Maui foi inicializado em `main`.

O estado tecnico `.maui/` pode ser versionado quando contiver orientacoes, placeholders e metadados tecnicos estaveis. Artefatos runtime, como logs gerados, devem permanecer fora do versionamento salvo decisao explicita futura.
